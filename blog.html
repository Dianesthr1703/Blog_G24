¿Cómo Empezar a Usar Git para Gestionar tus Versiones?

Git es una herramienta muy útil para cualquier persona que esté aprendiendo a programar, ya que nos ayuda a organizar y guardar nuestro trabajo de forma ordenada. En esta guía, te explicaré cómo usar Git desde cero, empezando por la creación de un repositorio local hasta la conexión con un repositorio remoto en GitHub.

Para crear un repositorio local en tu máquina, sigue estos pasos:

  1. Abre tu terminal o Git Bash. Puedes usar el atajo de teclado Win + R, escribir cmd y presionar Enter

  2. Verificar la versión de Git 
    $ git --version

    Este comando verifica la versión de Git instalada en tu sistema.

  3. Configurar el nombre de usuario y el correo electrónico 
    $ git config --global user.name "Diane"
    $ git config --global user.email "dianae2026@gmail.com"

    Configura tu nombre y correo electrónico para los commits.

  4. Navegar al directorio del proyecto 
    $ cd /c/Users/Diana/OneDrive/Documentos/Página\ web

    Cambia al directorio donde tienes tu proyecto.

  5. Inicializar el repositorio Git 
    $ git init

    Inicializa un nuevo repositorio Git en el directorio actual.

  6. Listar archivos en el directorio 
    $ ls

    Muestra los archivos en el directorio actual.

  7. Crear un nuevo archivo HTML 
    $ touch blog.html

    Crea un nuevo archivo llamado blog.html.

  8. Agregar el archivo al área de preparación 
    $ git add blog.html

    Agrega blog.html al área de preparación.

  9. Confirmar los cambios 
    $ git commit -m "Agregar blog.html"

    Realiza un commit de los cambios agregados.

  10. Crear y cambiar a una nueva rama 
    $ git checkout -b nueva-funcionalidad

    Crea y cambia a una nueva rama llamada nueva-funcionalidad.

  11. Agregar todos los cambios al área de preparación 
    $ git add .

    Agrega todos los archivos al área de preparación.

  12. Confirmar los cambios en la nueva rama 
    $ git commit -m "Nuevos cambios"

    Realiza un commit de todos los cambios en la nueva rama.

  13. Volver a la rama principal 
    $ git checkout main

    Cambia de nuevo a la rama principal.

  14. Fusionar la rama nueva con la rama principal 
    $ git merge nueva-funcionalidad

    Fusiona los cambios de nueva-funcionalidad en la rama principal.

  15. Agregar un repositorio remoto en GitHub 
    $ git remote add origin git@github.com:Dianesthr1703/Blog_G24.git

    Conecta tu repositorio local a tu repositorio en GitHub.

  16. Subir los cambios al repositorio remoto 
    $ git push -u origin main

    Envía los cambios a tu repositorio en GitHub.

Nota:

Recuerda siempre realizar commits frecuentes para mantener un buen historial de cambios y facilitar la colaboración.

¡Listo para comenzar a usar Git!

Con estos pasos, ahora deberías tener una idea clara de cómo inicializar un repositorio local, crear ramas y realizar commits. Git es una herramienta poderosa que te ayudará a gestionar tu trabajo de manera efectiva. ¡Explora y practica!
